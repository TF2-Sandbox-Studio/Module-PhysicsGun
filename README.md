# Module-PhysicsGun
Brings physics gun feature to tf2! (Sandbox version)

Command: `sm_physgun`

| Classname         | Supported     |
| -------------     |:-------------:|
| prop_dynamic      | ✓             |
| prop_physics      | ✓             |
| prop_ragdoll      |               |
| player            |               |
| tf_dropped_weapon | ✓             |
